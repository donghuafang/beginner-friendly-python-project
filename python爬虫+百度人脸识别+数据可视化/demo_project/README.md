项目概述
该项目实现了一个“人脸颜值分析工具”。其主要工作流程如下：

1. 用户输入 : 通过一个简单的 GUI 界面，用户可以输入一个 关键字 （例如“明星”）和希望爬取的 图片数量 。
2. 图片爬取 : 程序使用 icrawler 库，根据用户输入的关键字，从必应（Bing）图片搜索引擎上分别爬取 红色、蓝色、绿色、黑色 四种色调的图片。
3. 人脸分析 : 遍历所有下载的图片，调用 百度AI开放平台的人脸识别API ，对每张图片进行人脸检测，并获取“颜值”分数。
4. 图片分类 : 根据返回的颜值分数（1-10分，或-1表示未检测到人脸），将图片移动到对应的分数文件夹内。
5. 数据可视化 : 使用 pyecharts 库，将四种颜色风格下不同颜值分数的图片数量，分别生成 柱状图、折线图、饼图和散点图 。
6. 报告生成 : 将所有图表整合到一个名为 数据大屏.html 的网页中，并自动在浏览器中打开，形成一个可交互的数据可视化报告。

运行前注意：
1.安装所有项目所需的库（包括百度的ai模块baidu-aip）
2.在百度智能云中申请获取人脸识别所需的APP_ID,API_KEY和SECRET_KEY并将其填入到代码中的“三码部分”
